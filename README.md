Please cite as following:

Zsebők, S., Blázi, G., Laczi, M., Gergely N., Vaskuti É. and Garamszegi L.Z. 2018. 
“Ficedula”: an open-source MATLAB toolbox for cutting, segmenting and computer-aided clustering of bird song
